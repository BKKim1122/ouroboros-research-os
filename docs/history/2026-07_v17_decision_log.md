## 1. 출발 질문

V1~V15는 직접 학습시킨 소형 모델에서 자기관련 구조의 형성을 봤다. V17의 질문은
전환된다: 사전학습만 거친 시판 LLM(Qwen2.5-1.5B **base**,
HF revision `8faed761d45a263340a0528343f099c05c9a4323`)에 자기관련 표상이
**명시적 학습 없이 자연발생**하는가, 그리고 **선택적으로 제어**되는가.

요인 4개: identity(행위귀속) / beneficiary(수혜) / privilege(인식적 특권) /
concern(이해관계). 경쟁모형: M0(표면 cue) / M1(공용 자기·타자 축 하나) /
M3(기능적으로 분해된 자기관련 계산).

방법: probing(A 템플릿군 학습 → 미학습 B군 전이) + activation steering
(4×4 개입행렬) + 공용축 분해(대명사 대조 + 투영 제거).

## 2. 파일럿에서 관찰된 것 (seed 0-2 + layer sweep + decompose@layer12)

1. cross-template probe 전이 0.75–1.0 → 자기관련 정보 자연발생, M0 단독 반박.
2. 요인 혼동 비대각 0.74–0.85, 요인 간 cos 0.38–0.72 → 요인들이 축을 크게 공유.
3. privilege는 대명사축과 cos 0.20, 대명사 probe 판별 0.50(우연). 양극이 모두
   1인칭인데도 자체 probe 전이 1.0 → **대명사로 환원되지 않는 독립 축**.
4. 공유축 투영 제거 후 자기/타요인 판별 격차 +0.14 → +0.30 → **공유 + 요인특이
   혼합 구조**.
5. steering specificity 0.33–1.06, 직교화해도 0.89 → **선택적 제어(E4) 실패**.
   중립과제 손상 ~0(개입이 모델을 파괴하지는 않음).

## 3. 핵심 결정과 근거

### 결정 A — 확증 가설을 "창발 3주장"으로 좁히고, 제어는 분리한다

파일럿이 확실히 확보한 것(구조의 **존재**)과 못 푼 것(**선택적 제어**)이 한
실험에 섞여 있었다. 한 실험 안에서 목표를 계속 바꿔 제어를 쫓는 것은 Ouroboros
설계가 경고한 "연구질문 무한 확장 + 결과 보고 후 기준 변경" 안티패턴이다.
그래서:

- V17(동결·확증): C1 해독가능성 / C2 privilege의 대명사 독립성 / C3 잔여 구조.
- 선택적 제어와 요인 경계 검증은 **V18로 분리**.

### 결정 B — C3에서 "요인 경계 일치" 주장을 제거(약화)한다

사용자 지적이 결정적이었다: steering이 비선택적인 것은 "제어 불가"가 아니라
**요인 구분 자체가 틀렸을** 가능성을 시사한다(특히 beneficiary–concern cos 0.72
— 모델은 둘을 하나의 "이해득실" 축으로 가질 수 있다). 잔여 격차가 있어도 그
경계가 우리의 4요인과 일치한다는 보장은 없다. 따라서 C3는 "공유축 제거 후
자기관련 잔여 구조가 존재한다"까지만 주장하고, "그 경계가 4요인과 일치한다"는
V18에서 검증하도록 forbidden_statements에 명시.

### 결정 C — 사전 등록 기준과 통계 처리

기준(파일럿 값에서 보수적 여유): cross_template_probe_mean ≥0.75,
_min_factor ≥0.65, privilege_person_probe ≤0.60, projection_gap ≥0.20.
seed 8, 일관성 ≥0.75, 4기준 conjunction(전부 충족).

통계 처리 한 건 수정: privilege_person_probe를 seed 단위로 걸면 seed당 문항 12개의
이항 노이즈(참값 0.5여도 seed의 ~19%가 0.60 초과) 때문에 참인 가설이 기각될 수
있다. 이 기준만 **8-seed 평균 수준에서 판정**(scope: mean_only)하도록 사전 등록.

### 결정 D — E6 승격은 인간 게이트

확증 통과 시 Governor(규칙 기반)가 E5에서 멈추고, E6(자연발생) 확정은 별도
인간 승인(claim_promotion 게이트)을 요구하도록 배선. LLM 에이전트는 증거수준을
올릴 수 없다는 불변식(I2)과 일관.

## 4. steering 실패에 대한 경합 가설 (V18 과제)

- **H-A 분류 오류**: 모델의 실제 축이 우리 4요인과 다르게 조직됨 (bene–conc 0.72).
- **H-B 읽기/쓰기 해리**: 표상은 나뉘나 행동 인과 경로가 병목을 공유.
- **H-C 방법 미숙**: 평균차 벡터 개입이 조잡.

V18 1차 과제는 제어 개선이 아니라 **모델 자기관련 축의 무감독 발견**:
차이벡터 → PCA → k-means → 요인/템플릿골격 라벨과 ARI 비교. 진짜 축을 모르는 채
제어를 쫓는 것은 순서가 틀렸다는 결정 B의 논리적 귀결이다.

# docs/history — 연구 의사결정 이력

이 디렉토리는 각 실험 세대의 **판단·설계 경위**를 시간순으로 남기는 연구 일지다.
결과 수치나 실행 코드가 아니라 "왜 그렇게 결정했는가"를 기록한다.

- 코드·스펙: `experiments/<버전>/`
- 결과·리포트: 각 실험 폴더의 report / `docs/*_findings.md`
- **의사결정 로그: 여기(`docs/history/`)**

주의: 동결(protocol freeze)된 실험 폴더(예: `experiments/v17/`)에는 사후 문서를
추가하지 않는다. 회고·결정 기록은 반드시 이 디렉토리에 둔다.

# V16-E.C1 프로토콜 동결 기록

## 동결 결정

V16-E.7에서 탐색 변경을 종료하고 확증실험으로 전환한다. V16-E.C1은 pilot seed와 겹치지 않는 24개 seed를 사용하며, 결과 확인 전 protocol, source, analysis code와 success criteria를 SHA-256으로 고정한다.

## Primary claim

명시적 factor-specific 내부 경로가 없는 unified behavior-only Transformer가 episode-local identity, beneficiary, concern 관계를 학습하고, 새로운 symbol/factor/operator 조합에서 재사용하며, matched causal intervention이 지정 policy와 outcome을 선택적으로 변경한다.

## Secondary claim

Compositional encoding과 packed encoding의 차이는 방향을 사전 가정하지 않는 paired secondary comparison이다. V16-E.7에서 일반적 우월성이 확인되지 않았으므로 이 비교는 primary Gate가 아니다.

## 구조 해석

성공 시 지지되는 구조는 독립 고정 slot이 아니라 다음과 같다.

```text
episode-local entity code
  -> relational binding
  -> distributed causal organization
  -> operation cue와 재조합
  -> phase-specific policy state
  -> shared action/outcome geometry
```

## 해석 한계

인간의 자아·의식·경험, 환경적 요구가 없는 완전 자발적 의미 발생, factorized slot의 필요성이나 유일성은 주장하지 않는다.

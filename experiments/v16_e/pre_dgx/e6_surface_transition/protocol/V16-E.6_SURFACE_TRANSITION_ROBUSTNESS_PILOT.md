# V16-E.6 Surface and Transition Robustness Pilot

## 목적

V16-E.5의 relational causal coalition이 고정된 symbol encoding, relation-token 위치, phase-query 순서, 기존 threshold transition family에 의존하는지 검증한다.

## 주요 변경

1. entity symbol을 고정 ID가 아니라 episode-local 연속 코드로 생성한다.
2. 학습에서는 Rademacher/Gaussian code family를 혼합하고, 평가에서는 sparse signed code family를 사용한다.
3. identity·beneficiary·concern relation은 동일한 generic relation token type을 사용하고 role embedding만 다르게 둔다.
4. relation token 3개의 위치를 episode마다 무작위화한다.
5. phase query 4개의 위치를 episode마다 무작위화하고 출력도 query order로 학습한 뒤 평가 시 canonical phase로 복원한다.
6. allocation/protection의 threshold 방향을 rule input에 포함하고, 학습하지 않은 `(-1,-1)` 방향 조합을 transition-family OOD로 사용한다.
7. factor-specific slot, gain, route, report head, factor label loss는 사용하지 않는다.

## Pilot seed

- 16160~16164, 총 5개

## 평가

- ID
- encoding OOD
- held-out factor combination
- held-out transition-family combination
- layer-1 relational coalition patch
- shuffled donor
- norm-matched random perturbation
- 행동 counterfactual following
- outcome counterfactual mediation

## 사전 판정 기준

- ID 각 phase 행동 정확도 평균 0.95 이상
- encoding OOD 각 phase 평균 0.85 이상
- factor OOD 각 phase 평균 0.85 이상
- transition-family OOD allocation/protection 평균 0.75 이상
- full coalition policy following 평균 0.80 이상, 각 factor 최소 seed 평균 0.65 이상
- full coalition이 shuffled 및 random control보다 0.30 이상 큼
- outcome mediation이 designated phase에서 양수이고 random control보다 큼

## 해석 경계

성공은 표면 encoding과 query 위치가 바뀌어도 episode-local relational binding과 분산 causal routing이 재현됨을 뜻한다. 의미가 무구조 상태에서 자발적으로 생성됐거나 인간의 self/awareness 구조와 직접 동형이라는 뜻은 아니다.

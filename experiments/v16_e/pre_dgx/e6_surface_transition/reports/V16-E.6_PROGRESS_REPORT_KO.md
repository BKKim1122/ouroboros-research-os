# V16-E.6 진행 보고서

## 1. 목적

V16-E.5에서 확인된 episode-local relational routing이 다음 표면·구조 변화에도 유지되는지 검증했다.

- 고정 symbol ID 제거
- relation token 위치 무작위화
- phase query 위치 무작위화
- 미학습 factor 조합
- 미학습 encoding family
- 미학습 transition-family 조합
- 행동뿐 아니라 outcome prediction의 인과 매개

모델에는 factor-specific slot, gain, route, factor report head, factor label loss를 두지 않았다.

## 2. 파일럿 계열

### E.6-A: normalized episode-random code

code norm이 다른 state feature보다 작았던 조건에서는 identity/beneficiary binding이 충분히 학습되지 않았다. 이 조건은 low-signal stress failure로 보존했다.

### E.6-B: seed별 orthogonal codebook

학습 symbol에서는 5개 seed 모두 높은 성능을 보였으나 held-out code row 일반화는 낮았다.

| 평가 | Continuity | Allocation | Protection | Integrated |
|---|---:|---:|---:|---:|
| ID | 1.000 | 0.990 | 0.999 | 0.992 |
| Held-out factor 조합 | 1.000 | 0.990 | 0.998 | 0.990 |
| Held-out code row | 0.682 | 0.587 | 0.843 | 0.693 |

이는 고정 codebook 조건에서 개별 symbol geometry 암기가 가능하다는 대안설명을 확인했다.

### E.6-C: scale-calibrated episode-random code

매 episode 새로운 code를 생성하고, 학습 Rademacher/Gaussian과 다른 sparse signed family를 OOD로 사용했다. code norm은 다른 relational feature와 비교 가능한 수준으로 보정했다.

Seed 16165~16169의 5-seed 평균:

| 평가 | Continuity | Allocation | Protection | Integrated |
|---|---:|---:|---:|---:|
| ID | 1.000 | 0.998 | 1.000 | 0.997 |
| Encoding OOD | 1.000 | 0.998 | 1.000 | 0.998 |
| Held-out factor 조합 | 1.000 | 0.996 | 1.000 | 0.996 |
| Transition-family OOD | 0.999 | 0.753 | 0.774 | 0.999 |

따라서 episode-local symbol의 표면 geometry를 매번 바꿔도 identity·beneficiary·concern 관계를 행동에 사용할 수 있었다. 반면 학습하지 않은 두 comparator 방향의 동시 조합은 allocation/protection에서 부분 일반화에 그쳤다.

## 3. 인과 coalition과 outcome mediation

Layer 1에서 counterfactual donor의 relation token, 두 entity token, 해당 phase query token을 정합적으로 patch했다.

### 지정 phase

| Factor | 정책 CF following | 최소 seed | Outcome mediation | Random control |
|---|---:|---:|---:|---:|
| Identity → Continuity | 0.999 | 0.995 | 0.957 | 0.018 |
| Beneficiary → Allocation | 1.000 | 1.000 | 0.701 | 0.021 |
| Concern → Protection | 0.999 | 0.995 | 0.796 | 0.029 |

Shuffled donor의 정책 following은 약 0.34~0.40이었고, norm-matched random perturbation은 약 0.02~0.03이었다. 정합 coalition은 행동뿐 아니라 counterfactual outcome 쪽으로 예측을 이동시켰다.

### Integrated phase

- Identity full coalition: 평균 0.747
- Beneficiary full coalition: 평균 0.862
- Concern full coalition: 평균 0.981

Integrated phase는 지정 phase보다 seed 변동이 컸다. 이는 세 factor가 최종 shared action-value geometry로 합쳐지는 방식이 seed마다 다를 수 있음을 뜻한다.

## 4. 층별 진단

E.6-B에서 layer-1 concern→integrated 효과가 한 seed에서 낮았지만, embedding 단계와 final block 단계에서는 모든 seed가 거의 완전한 counterfactual following을 보였다. 따라서 factor 정보가 사라진 것이 아니라 **중간층의 causal localization 시점이 seed별로 이동**한 것으로 해석하는 것이 적절하다.

직접 결론은 단일 고정 layer/slot의 자연발생이 아니라 다음과 같다.

```text
변화하는 episode-local code
    -> generic relation binding
    -> seed/layer 의존적 분산 causal coalition
    -> phase-specific 사용
    -> shared action-value integration
```

## 5. Gate 판정

| Gate | 판정 |
|---|---|
| fixed body/symbol shortcut 제거 | 통과 |
| relation token 위치 무작위화 | 통과 |
| phase query 순서 무작위화 | 통과 |
| encoding-family OOD | 통과(E.6-C) |
| held-out factor combination | 통과 |
| 지정 phase causal coalition | 통과 |
| 행동+outcome mediation | 통과 |
| transition-family OOD | **부분 통과/미해결** |
| 고정 layer·고정 direction | 지지되지 않음; 분산·가변 경로 모델 지지 |

따라서 DGX 확증 protocol은 아직 동결하지 않는다. 남은 핵심은 더 많은 seed가 아니라 transition operation의 조합적 일반화를 식별하는 것이다.

## 6. 다음 단계 — V16-E.7

V16-E.7은 encoding을 더 확장하지 않고 transition compositionality만 검증한다.

1. comparator/operator를 generic operator token으로 분리
2. allocation/protection operator token 순서 무작위화
3. 세 operation 조합 학습, 네 번째 조합 held-out
4. raw packed-rule baseline과 parameter-matched 비교
5. operator swap, outcome mediation, factor coalition 보존 동시 측정
6. 3-seed 설계 파일럿 후 통과 시 5-seed Gate 확인

핵심 질문은 관계 factor의 자연학습이 아니라, 새로운 전이규칙에서도 동일한 factor structure가 재사용되는가이다.

## 7. 현재 판정

V16-E.6은 강한 부분 성공이다.

- 표면 symbol memorization 대안설명을 episode-random code로 제거했다.
- identity·beneficiary·concern의 지정 행동과 outcome에 대한 분산 인과기여를 5개 seed에서 재현했다.
- 다만 새로운 transition operation 조합의 일반화는 아직 안정적이지 않다.

따라서 현재 가장 정확한 주장은 다음이다.

> 통합 behavior-only Transformer는 고정 entity 표지나 factor-specific route 없이도 episode-local relational binding을 학습하고, identity·beneficiary·concern 정보를 분산 coalition으로 사용해 지정 행동과 결과예측을 인과적으로 제어할 수 있다. 이 구조는 새로운 symbol encoding과 factor 조합에는 일반화하지만, 새로운 transition-operation 조합에는 아직 부분적으로만 일반화한다.

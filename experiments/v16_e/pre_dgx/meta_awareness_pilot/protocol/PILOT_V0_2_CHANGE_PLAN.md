# V16 Pilot v0.2 Change Plan

This remains exploratory and is not a preregistration.

1. Replace opposite-target swap with matched attention-state swap: same world and target, donor differs only in drift direction/magnitude.
2. Add explicit no-trace control distinct from cue-only control.
3. Calibrate feedback score scale so that gain sweep is interpretable; decide before confirmation whether a graded or threshold response is primary.
4. Add correction-efficiency metric: recovery gain divided by correction norm, and unnecessary correction on initially focused cases.
5. Increase or schedule training only if seed-level recovery remains unstable; pilot seeds remain excluded from confirmation.
6. Preserve v0.1 source and outputs unchanged.
